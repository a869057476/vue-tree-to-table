# vue-tree-to-table
树形结构转table

## Getting started

```bash
# clone the project
git clone https://github.com/a869057476/vue-tree-to-table.git

# enter the project directory
cd vue-tree-to-table

# install dependency
npm install

# develop
npm run start
```

This will automatically open http://localhost:8080

# Build
npm run build
